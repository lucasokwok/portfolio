# Conceito do Portfólio — Lucas Okwok

## 1. Direção artística

**Nome do conceito:**  
**Corporate Tech Portfolio**

A ideia é criar um portfólio com aparência de **desenvolvedor full stack corporativo**, com foco em sistemas internos, automações, dados, dashboards e aplicações que resolvem problemas reais de empresa.

O site não deve parecer muito artístico, experimental ou cheio de efeitos. O visual ideal é:

```txt
Escuro
Profissional
Direto
Técnico
Confiável
Bom para recrutadores
Com aparência de produto corporativo/SaaS
```

A mensagem principal do site deve ser:

> Desenvolvedor Full Stack com experiência em aplicações web, automações corporativas, análise de dados e soluções internas voltadas à eficiência operacional.

---

## 2. Posicionamento profissional

O portfólio não deve vender apenas a ideia de um desenvolvedor que sabe React. O diferencial principal é mais amplo:

```txt
Desenvolvedor Full Stack que conecta software, dados e automação para resolver problemas operacionais reais.
```

Esse posicionamento combina com os principais projetos e experiências:

```txt
- Aplicativos corporativos
- Power Apps
- Mapas e inspeções
- Automações SAP
- Dashboards Power BI
- Python para dados e automações
- React/TypeScript/Vite para web apps
```

Portanto, o site deve mostrar que Lucas Okwok é um profissional que entende **produto, processo, dados e desenvolvimento**.

---

## 3. Frase principal do Hero

### Opção 1

```txt
Desenvolvedor Full Stack focado em criar aplicações web, automações e soluções baseadas em dados para otimizar processos corporativos.
```

## 6. Projetos em destaque

Os projetos devem ser apresentados como **cases corporativos**, não como projetos pequenos ou acadêmicos.

### Ordem ideal dos projetos

```txt
1. Hunter Map
2. SAP Automations
```

### 6.1 Hunter Map

Descrição:

```txt
Aplicação web de navegação em mapa para apoio a inspeções em campo. O sistema exibe pontos de interesse suspeitos indicados por modelo de aprendizado de máquina e permite registrar inspeções, classificando resultados como fraude ou falso positivo.
```

Tags:

```txt
React
TypeScript
Vite
Maps
Machine Learning
Inspection
```

Observação:

```txt
Esse deve ser tratado como o projeto mais forte do portfólio.
```

---

### 6.2 SAP Automations

Descrição:

```txt
Aplicativo desktop em Python com Tkinter para centralizar automações no SAP. A ferramenta permite selecionar jobs, importar dados via planilhas XLSX, configurar execuções e gerar relatórios completos de saída.
```

Tags:

```txt
Python
Tkinter
XLSX
SAP
Automation
Reports
```

```txt
Aplicativo de reservas de vagas de estacionamento para ambiente corporativo, desenvolvido em Power Apps. A solução permite organizar reservas, controlar disponibilidade e facilitar a gestão interna de vagas.
```

Tags:

```txt
Power Apps
```
