# PROJECT CONTEXT

## Objetivo
Reorganizar o portfólio pessoal de Lucas Okwok em uma base única, limpa e pronta para desenvolvimento e deploy na Vercel, preservando a identidade visual escolhida.

## Stack utilizada
- React
- TypeScript
- Vite
- Tailwind CSS 4
- Lucide React

## Direção visual
- Tema escuro corporativo
- Aparência de produto profissional/SaaS
- Layout direto, técnico e escaneável
- Sem animações ou efeitos decorativos que mudem a identidade aprovada

## Estrutura principal do site
- Header fixo com navegação, GitHub, LinkedIn e toggle PT/EN
- Hero com nome, função, foto/placeholder, CTA e métricas
- Sobre
- Projetos
- Tecnologias
- Experiência
- Formação
- Contato
- Footer

## Projetos que devem aparecer no portfólio
1. Hunter Map
2. SAP Automations
3. Minha Primeira Automação
4. Parking App
5. Painel Power BI de Gestão de Metas

## Links principais
- LinkedIn: https://www.linkedin.com/in/lucasokwok
- GitHub: https://github.com/lucasokwok

## Decisões importantes
- A versão final escolhida foi tratada como base única.
- O projeto foi migrado para Vite para simplificar o deploy na Vercel.
- Os textos principais foram mantidos, com organização em arquivos de dados estáticos.
- O suporte PT/EN permanece local e simples, baseado em objeto de tradução.
- As versões antigas e os artefatos do template anterior devem ser removidos.
- A referência de conceito original foi preservada em `docs/conceito_portfolio_lucas_okwok_atualizado.md`.
