# Conceito do Portfólio — Lucas Okwok

## 1. Direção artística

**Nome do conceito:**  
**Corporate Tech Portfolio**

A ideia é criar um portfólio com aparência de **desenvolvedor full stack corporativo**, com foco em sistemas internos, automações, dados, dashboards e aplicações que resolvem problemas reais de empresa.

O site não deve parecer muito artístico, experimental ou cheio de efeitos. O visual ideal é:

```txt
Escuro
Profissional
Direto
Técnico
Confiável
Bom para recrutadores
Com aparência de produto corporativo/SaaS
```

A mensagem principal do site deve ser:

> Desenvolvedor Full Stack com experiência em aplicações web, automações corporativas, análise de dados e soluções internas voltadas à eficiência operacional.

Esse posicionamento combina com os principais projetos e experiências:

- Power Apps
- Mapas e inspeções

Portanto, o site deve mostrar que Lucas Okwok é um profissional que entende **produto, processo, dados e desenvolvimento**.

# Projeto adicional — Minha Primeira Automação

## Por que adicionar este projeto ao portfólio

Este projeto deve ser incluído no portfólio porque mostra uma competência muito valorizada por recrutadores: a capacidade de identificar um problema real, entender um processo manual, propor uma solução simples e evoluir essa solução para um software mais completo.

Ele também ajuda a construir uma narrativa profissional forte, porque mostra claramente a transição entre:

```txt
Problema manual → Automação simples → Produto interno personalizado
```

Além disso, esse projeto comunica bem que o desenvolvimento não começou apenas como estudo técnico, mas como solução para uma dor real de uma pessoa e de um ambiente profissional.

## Como apresentar no portfólio

Nome sugerido do case:

```txt
Minha Primeira Automação
```

ou, em uma versão mais profissional:

```txt
Legal Office Automation
```

ou:

```txt
Automação de Cartas Jurídicas
```

A melhor opção para o portfólio em português é:

```txt
Minha Primeira Automação
```

porque ela cria uma narrativa mais humana e memorável.

## Descrição narrativa

```txt
Minha primeira automação nasceu a partir de uma necessidade real observada em um escritório jurídico. Em uma conversa, minha namorada, que trabalha em um escritório de advocacia e cursa Direito, comentou que uma vez por mês precisava realizar uma tarefa extremamente manual: gerar e enviar cartas para unidades de condomínios.

O processo exigia seguir um modelo de carta previamente definido e substituir manualmente informações como nome do responsável pela unidade, identificação da unidade, número do apartamento, valores em aberto e outros dados específicos de cada notificação.

Ao analisar o processo, percebi que ele poderia ser automatizado. A primeira versão da solução permitia importar uma planilha com todas as unidades a serem notificadas, contendo as informações necessárias para preenchimento do modelo. A partir desses dados, o sistema gerava automaticamente um arquivo DOCX e um PDF individual para cada unidade.
```

## Descrição resumida para card de projeto

```txt
Automação criada para gerar cartas jurídicas em lote a partir de um modelo e uma planilha XLSX, produzindo automaticamente arquivos DOCX e PDF individualizados para cada unidade notificada.
```

## Evolução do projeto

```txt
Com o tempo, a automação evoluiu para um software desktop personalizado para o escritório jurídico. A solução passou a centralizar dados de condomínios clientes, unidades, contatos e processos de cobrança, além de extrair e integrar relatórios de softwares utilizados por administradoras de condomínios.

O sistema também passou a automatizar o envio de cobranças por e-mail e a geração de cartas, mantendo como base a ideia inicial que deu origem ao projeto: reduzir tarefas repetitivas, padronizar documentos e economizar tempo em processos administrativos jurídicos.
```

## Descrição atual do projeto

```txt
Software desktop personalizado para escritório jurídico, voltado à automação de cobranças condominiais, geração de documentos, organização de dados de unidades e integração com relatórios de administradoras.
```
